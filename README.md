# ColorCollection
react+redux+typescript+express+mongoで作成したサイト
好きな配色をストックすることができます。

# 開発
- フロント
```npm start```

- バックエンド
```npm start```

- DB
```brew services start mongodb-community```

# 使用技術
- フロントエンド
  - react
  - react-hooks
  - react-router
  - redux
  - styled components
- サーバーサイド
  - node.js
  - express
- DB
  - mongoDB
